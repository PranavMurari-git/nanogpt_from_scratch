# NanoGPT from Scratch

A lightweight, decoder-only Transformer model built entirely from scratch in PyTorch. This project demonstrates the fundamental mathematical architecture behind modern Large Language Models (LLMs).

## Project Overview
Instead of fine-tuning an existing model or calling an API, I built the core components of a Generative Pre-trained Transformer from a blank Python script. The model was trained on the TinyShakespeare dataset to generate character-level text.

### Technical Achievements
* **Architecture:** Implemented Scaled Dot-Product Self-Attention, Multi-Head Attention, and Feed-Forward networks natively in PyTorch.
* **Optimization:** Utilized `AdamW` optimizer and engineered residual connections with Layer Normalization to ensure stable gradient flow during training.
* **Results:** Successfully trained a [42,000] parameter model. Through the implementation of the Self-Attention mechanism, the model successfully reduced cross-entropy loss from a baseline of [4.37] to [1.89], learning the structural syntax and vocabulary of Shakespearean dialogue.

## The Architecture
The model relies on the following custom-built PyTorch modules:
1. **Character-Level Tokenizer:** Converts raw text into integer sequences.
2. **Positional Encodings:** Injects spatial awareness into the embedding space so the model understands token order.
3. **Causal Masking:** Ensures the self-attention mechanism strictly predicts the future without "cheating" by looking ahead.
4. **Transformer Blocks:** Combines communication (Attention) with computation (Feed-Forward).
